Windows Phone

The NYTimes app allows you to enjoy the award-winning journalism of The New York Times on your Windows Phone, wherever you are. The app has been specifically designed for browsing and navigating articles, features and video on the Windows Phone platform. Download and sync the latest news directly to your device via Wi-Fi or cellular network to take it with you wherever you go. Share articles, photos and videos with your friends through Facebook, Twitter and e-mail and be the first to read about events as soon as they occur with breaking news alerts.

Available in the Windows Marketplace for Mobile.

Note: This FAQ applies to the NYTimes app for Windows Phone 7.5 or later. If you are using Windows Phone 7, you can find more information on the NYTimes app in our Windows Phone 7 FAQ.

  • 1. What features does the NYTimes app for Windows Phone have?
    Back to top

    The NYTimes application for Windows Phone allows you to browse articles, blogs, videos and more. You can save articles and pin sections for easy access.

  • 2. Does it cost anything to use the application?
    Back to top

    The NYTimes application is free to download and install, and includes some free content. However, to get unlimited access to all sections, you must become a digital subscriber. For more information, please see the Digital Subscriptions and Premium Products FAQ.

    Please note that in the absence of a Wi-Fi connection, the NYTimes app uses your phone's cellular network connection to access and download content. Please check with your carrier to see whether you will be charged for usage, especially while traveling.

  • 4. How do I get the NYTimes app?
    Back to top

    The NYTimes app for Windows Phone is available in the Windows Marketplace for Mobile. By default, you can access the Marketplace from the Start page of your phone.

    Back to top

    You can navigate to the Saved section by flicking over from any page in the main area of the app. From Saved, you can view your saved articles. For more information, see "How do I save an article?"

  • 15. How do I remove the NYTimes application from my Windows Phone?
    Back to top

    To remove the NYTimes application, follow the same procedure that you'd use for any Windows Phone app. From the Windows Phone Start page, flick to the App list. Tap and hold the NYTimes application and choose uninstall.

  • 16. Do I need to register for NYTimes.com to use the NYTimes app?
    Back to top

    No. You can download the NYTimes application and access some content without registering. However, full access requires registration and a subscription. If you have already registered with NYTimes.com, you can use the same user name and password to log in to the app. For full access, you must subscribe to a digital package that includes smartphone apps.
